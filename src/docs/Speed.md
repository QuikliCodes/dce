# Speed

Speed is one of the four [Attributes](Attributes.md).  It is used to determine how
fast a character can move, what order they take in initiative, and for their
ability to dodge attacks.

Examples:

- Dodge an attack
- Flying or teleportation boons
- Move at speed
- Determine order in [initiative](Initiative.md)

## Movement Speed

**Calculation:** 2 + Speed attribute score

Movement speed is the distance a character can move in a single [move action](Turn Actions.md#move-actions)
